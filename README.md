# House Price Prediction (Regression)

## Project Overview
This project focuses on predicting house prices using Machine Learning regression algorithms. Accurate house price prediction helps buyers, sellers, and real estate professionals make informed decisions.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation to identify the best-performing regression model.

---

## Dataset
The dataset contains housing-related features used to predict house prices.

**Target Variable:**  
- House Price

**Key Features Include:**
- Number of Bedrooms
- Number of Bathrooms
- Square Footage (Area)
- Location
- Year Built
- Number of Floors
- Garage Availability
- Lot Size
- House Condition
- Other Property Features

## Project Workflow

### Data Preprocessing
- Checked missing values
- Removed duplicates
- Handled outliers
- Encoded categorical variables
- Applied feature scaling

### Exploratory Data Analysis (EDA)
- Analyzed feature distributions
- Visualized relationships between features
- Identified key factors affecting house prices

### Model Building
The following regression models were implemented:

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- K-Nearest Neighbors (KNN) Regressor

### Model Evaluation
Models were evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## Results

Model evaluation metrics:

- **Mean Absolute Error (MAE):** 8174.58  
- **Mean Squared Error (MSE):** 10,143,798.51  
- **Root Mean Squared Error (RMSE):** 10,071.48  
- **R² Score:** **0.9984** ✅  

**Best Model Performance:**  
The model achieved an **R² Score of 0.9984**, indicating very high prediction accuracy and strong model performance.
